대부분의 컴퓨터공학 용어는 실제로 자주 쓰는 언어다.
TAG 가장 지배적인 문법, TAG는 설명한다.
_ 열리는 태그 , / 닫히는 태그

/strong : 굵게.
/u (underline) : 밑줄.
/h1,2,3,4,5,6 (heading) : 제목 - 굵게, 1이 제일 굵음, 줄바꿈.
br (line breaker) : 줄바꿈. 닫지 않는다.
/p (paragragh) : 단락바꿈.
img : 이미지
img src="dogs.jpg" width="200" > attribute 속성 - 순서는 상관없음.
ctrl + 여러곳 클릭 > 클릭한 곳들에 동시에 같은 문자 입력가능.

li는 자식tag, ul,ol은 부모tag. 2대 서로가 꼭 필요함.
/li (list) : 목차표시.
/ul (unordered list) : 어디부터 어디까지가 연관된 내용인지 표시. 자동으로 숫자 붙이지않음.
/ol (ordered list) : 자동으로 숫자가 붙음.

/title : 페이지 제목.
meta charset(문자규칙)="utf8" : 컴파일러야 utf8로 읽어줘. 글씨 깨지지 않도록.
/body : 본문을 구분하기 위한 규칙. 본문을 구성.
/head : 본문을 설명하는 것. 묶기로 규칙.
!doctype html , html : 이 모든 걸 묶는 것. '이 문서는 html입니다.'

html specification : html 설명서
a href="" , /a (hyper referende) : 링크 걸때.
target="_blank" : 링크를 눌렀을 때, 새 탭으로 띄운다.
title="" (tool tip): 링크 걸린 부분에 마우스를 댔을때, "" 라고 뜨게 한다. (설명)

hosting : 인터넷을 빌려주는 것
wed hosting : web을 사용하는데 필요한 소프트웨어까지 깔아서 빌려주는 것.
